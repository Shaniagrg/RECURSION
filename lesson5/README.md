<h3>Multiple Recursion</h3>

<ol>
    <li>Fibonacci number: Lets say you start with 0 and 1. a = 0 1</li>
    <li>1: so the next fibonacci number will be 0 + 1 = 1  OUTPUT = 0 1 1 </li>
    <li>2: so the next fibonacci number will be 1 + 1 = 2  OUTPUT = 0 1 1 2</li>
    <li>3: so the next fibonacci number will be 1 + 2 = 3  OUTPUT = 0 1 1 2 3</li>
    <li>4: so the next fibonacci number will be 2 + 3 = 5  OUTPUT = 0 1 1 2 3 5</li>
    <li>5: so the next fibonacci number will be 3 + 5 = 8  OUTPUT = 0 1 1 2 3 5 8 ......... and so on </li>
</ol>
