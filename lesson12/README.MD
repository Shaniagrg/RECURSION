<h3>Permutation</h3>
<p>A permutation refers to all the possible ways you can arrange a specific set of elements where the order doesn't matter</p>

<ol>
    <li>Concept the TC usually here is FACTORIAL !. lets breakdown with example</li>
    <li>2! = 2*1 = 2</li>
    <li>3! = 3 * 2 * 1 = 6</li>
    <li>5! = 5 * 4 * 3 * 2 * 1 = 120</li>
</ol>
